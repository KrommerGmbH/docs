# CMH Chatbot 

현재는 한국어(KO) 문서만 운영합니다.


## Concept 
cmh chatbot 은 오픈소스 프로젝트로 챗봇의 코어 기능을 담당합니다.

이 챗봇은 pnpm 또는 github 에서 다운로드하여 단독으로 사용할 수 있고, 원래 목적은 CMH 프로젝트 (shopware 6 plugin, aideworks electron app)의 일부로 CMH 프로젝트의 패키지로 프로젝트 전체 프로세스의 기능 중 일부를 담당합니다.
